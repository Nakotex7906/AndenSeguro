# Sistema de Monitoreo de Incidencias con IA

## Descripción del Proyecto
Este proyecto implementa un sistema de monitoreo inteligente basado en **Computer Vision** para la detección temprana de conductas de riesgo en estaciones de metro. El software analiza flujos de video en tiempo real para identificar patrones inusuales y alertar automáticamente a los equipos de emergencia.

### Objetivo General
Diseñar e implementar un sistema de monitoreo inteligente basado en cámaras de seguridad para la detección temprana y prevención de posibles intentos de suicidio, con el fin de alertar oportunamente a los equipos de emergencia y salvaguardar la vida de los transeúntes.
