
Sistema de Monitoreo de Incidencias

Objetivo General
Diseñar e implementar un sistema de monitoreo inteligente basado en cámaras de
seguridad para la detección temprana y prevención de posibles intentos de suicidio en las
estaciones de metro, con el fin de alertar oportunamente a los equipos de emergencia y
salvaguardar la vida de los transeuntes.
